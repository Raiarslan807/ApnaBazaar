# 🛒 ApnaBazaar – Sasta, Tez, Apna

ApnaBazaar is a modern, responsive Pakistani e-commerce storefront built using pure HTML, CSS, and JavaScript.

It provides a complete online shopping experience with category browsing, product search, flash sales, wishlist management, shopping cart functionality, and a checkout system.

## ✨ Features

- 🎨 Modern responsive UI
- 🛍️ Product categories
- 🔍 Search functionality
- ⚡ Flash sale section
- ❤️ Wishlist system
- 🛒 Shopping cart
- ➕ Add/remove quantity controls
- 💳 Checkout modal
- 🚚 Cash on Delivery support
- 📱 Mobile-friendly design
- 🇵🇰 Pakistani-themed branding

---

## 🛠️ Technologies Used

- HTML5
- CSS3
